Weekly Nutrition Comparison





Definiiton

  • For each graph, let
    • Nbreakfast, selected nutrition = the amount of selected nutrition provided by today's breakfast,
    • Nlunch, selected nutrition     = the amount of selected nutrition provided by today's lunch,
    • Ndinner, selected nutrition    = the amount of selected nutrition provided by today's dinner,
  • The length of each blue bar indicates Nbreakfast, selected nutrition,
  • The length of each orange bar indicates Nlunch, selected nutrition,
  • The length of each green bar indicates Ndinner, selected nutrition.

Operation

  • Select different nutritions by using the select menu.
  • Show the recipe name/amount of nutrition by moving mouse over a bar.
  • Show the grouped/stacked bar chart by clicking the radio buttons.
